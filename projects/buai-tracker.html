Buai Tracker

Android app for tracking betelnut consumption and spending habits in PNG

2024 Completed

About the Project

Buai Tracker is an innovative Android application designed specifically for betelnut (buai) chewers in Papua New Guinea. The app helps users monitor their chewing habits, track spending on betelnut, and maintain awareness of their consumption patterns.

As a culturally specific solution, Buai Tracker addresses the unique needs of PNG's betelnut consumers while promoting responsible consumption and financial awareness. The app provides valuable insights into spending patterns and helps users make informed decisions about their betelnut consumption.

Key Features

  • Expense Tracking: Monitor daily, weekly, and monthly spending on betelnut with detailed analytics
  • Consumption Analytics: Visualize chewing habits with interactive charts and consumption insights
  • Budget Management: Set monthly budgets and receive intelligent spending alerts
  • Health Reminders: Gentle notifications about consumption patterns and health considerations
  • Offline Capability: Full functionality without internet connection for remote areas
  • Multi-language Support: Available in Tok Pisin and English for wider accessibility
  • Data Export: Export consumption reports for personal record keeping

Technical Implementation

Technologies Used

Flutter Dart Firebase SQLite REST API Google Maps API

The app was built using Flutter for cross-platform compatibility, with Firebase handling backend services and user authentication. Local data storage using SQLite ensures offline functionality, while the clean architecture pattern maintains code scalability and testability.

Future Enhancements

  • Social Features: Community challenges and sharing capabilities among users
  • Vendor Integration: Connect with local betelnut vendors for direct purchasing
  • Health Insights: Integration with health monitoring devices and wellness tracking
  • Web Dashboard: Companion web application for detailed analytics and reporting
  • Gamification: Reward system for achieving responsible consumption goals
  • Marketplace: Built-in marketplace connecting buyers and sellers directly
