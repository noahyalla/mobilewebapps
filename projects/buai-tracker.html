Buai Tracker

Revolutionizing betelnut consumption tracking with an intuitive Android app designed for Papua New Guinea's unique market

🕒 2 Months ✅ Completed 📱 Mobile App
🎯 Problem Solver 💡 Innovative 🌍 Cultural Impact

Project Overview

Buai Tracker represents a groundbreaking approach to personal finance and health awareness in Papua New Guinea. This Android application specifically addresses the cultural significance of betelnut (buai) consumption while promoting responsible spending and health consciousness.

The app was born from observing the need for better financial tracking among betelnut consumers and the opportunity to leverage mobile technology for positive behavioral change. By combining expense tracking with consumption analytics, Buai Tracker empowers users with data-driven insights into their habits.

Key Features

  • Smart Expense Tracking: Real-time monitoring of daily, weekly, and monthly betelnut expenditures with intelligent categorization and spending alerts.
  • Insights: Interactive charts and visualizations showing consumption patterns, spending trends, and habit formation over time.
  • Offline-First Architecture: Complete functionality without internet connectivity, ensuring accessibility in remote areas of PNG.
  • Data Privacy & Security: Local data storage with optional cloud backup, ensuring user privacy and data protection.

Technical Implementation

Core Technologies

Flutter Dart Firebase SQLite REST API Provider Hive

Built with Flutter for optimal cross-platform performance, the app employs a clean architecture pattern ensuring scalability and maintainability. Firebase handles user authentication and optional cloud synchronization, while SQLite provides robust local data storage for offline functionality.

The application features a responsive design that adapts to various screen sizes, with particular attention to devices commonly used in Papua New Guinea. The state management is handled through Provider for efficient reactivity and performance optimization.

Future Roadmap

  • Community Features: Implement social sharing, community challenges, and peer support systems to enhance user engagement and accountability.
  • Vendor Network: Develop a comprehensive vendor directory with location-based services, reviews, and direct purchasing capabilities.
  • Health Integration: Partner with health organizations to provide educational content and integrate with wearable devices for comprehensive wellness tracking.
  • Web Platform: Create a companion web dashboard with advanced analytics, reporting tools, and data export capabilities for personal record-keeping.
  • Gamification System: Introduce achievement badges, rewards, and progress tracking to encourage consistent usage and goal achievement.
  • Marketplace Expansion: Develop an integrated marketplace connecting consumers directly with betelnut vendors for seamless transactions.
  • Budget Management System: Customizable budget limits with proactive notifications and savings recommendations based on spending behavior.
